1. Tenant-Wide Services Tab

This tab displays services that apply globally across your entire organization (tenant).

  • View: Shows a static table listing all common services, their current Status (Active/Inactive), Deployment status (Successful/Failed/Empty), and the Enable toggle switch.
  • Interaction: This view is primarily for information regarding shared service infrastructure.

2. Account-Specific Services Tab

This tab allows you to manage services for individual accounts. The input field functions as a powerful Select Box with Autocomplete to efficiently search through 2000 accounts.

2.1 Selecting an Account (Search-Select Feature)

  • Default Account: When you first click this tab, the system automatically loads the data for the first account (*Account-0001*) for immediate context.
  • View Closely Related Accounts (Dropdown Mode): If the search field is empty, click the input box. The system will automatically display the 5 accounts immediately following the currently selected account.
  • Search/Filter with Autocomplete (Recommended!):
    • Start Typing: Simply begin typing the account name or ID (e.g., Account-1500 or Marketing) directly into the select box.
    • Autocomplete will assist you by filtering the 2000 accounts in real-time.
    • Select: Click any account name in the suggestion list to instantly load its service data.

2.2 Managing Services

Once an account is selected, the dynamic table shows the service configuration:

ColumnDescriptionKey Values
ServiceThe name of the service.Varies
StatusThe current operational status.Active or Inactive
DeploymentThe result of the last configuration.Successful, Failed, or Empty
EnableToggle to turn the service On or Off for this account.Toggle switch

Action: Use the toggle switch in the Enable column to immediately activate or deactivate a service.
